Capture zone for all input methods. Process items regularly and move to appropriate folders.

## Subfolders

- **Quick/** - Fast text captures, fleeting thoughts
- **Voice/** - Voice recordings and transcriptions
- **Scanned/** - Handwritten notes, documents
- **Web/** - Web clips, saved articles

## Processing Rules

1. Review inbox daily or weekly
2. Tag appropriately
3. Link to relevant projects/areas
4. Move processed items to permanent locations
5. Archive or delete if no longer needed

## Quick Capture Templates

Use QuickAdd plugin with these shortcuts:

- `Ctrl+Shift+N` - Quick note
- `Ctrl+Shift+V` - Voice memo
- `Ctrl+Shift+T` - Task capture
