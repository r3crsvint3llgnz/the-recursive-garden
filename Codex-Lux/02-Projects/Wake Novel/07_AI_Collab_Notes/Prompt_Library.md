# Prompt Library

## Character Development Prompts

### Character Creation
```
Create a detailed character profile for [CHARACTER ROLE] in a science fiction novel. Include:
- Physical description and age
- Personality traits and flaws
- Background and motivation
- Speech patterns and mannerisms
- Character arc potential
- Relationships with other characters
Context: [Brief story context]
```

### Dialogue Generation
```
Write dialogue between [CHARACTER 1] and [CHARACTER 2] discussing [TOPIC].
Character 1 traits: [Key personality elements]
Character 2 traits: [Key personality elements]
Setting: [Location and situation]
Emotional tone: [Mood/atmosphere]
Story purpose: [What this dialogue should accomplish]
```

### Character Voice Consistency
```
Review this dialogue/scene for character voice consistency:
[PASTE TEXT]

Character guidelines:
- [CHARACTER NAME]: [Voice description]
- [CHARACTER NAME]: [Voice description]

Check for: Speech patterns, vocabulary, personality consistency, emotional authenticity
```

## World-Building Prompts

### Technology Development
```
Design a science fiction technology system for [TECHNOLOGY TYPE].
Requirements:
- Scientific plausibility within [UNIVERSE RULES]
- Story integration possibilities
- Limitations and consequences
- Cultural/social impact
- Visual/sensory description potential
Related to: [Story elements it affects]
```

### Environmental Description
```
Describe a [ENVIRONMENT TYPE] that serves the story by [PURPOSE].
Include:
- Sensory details (sight, sound, smell, texture)
- Unique characteristics that affect plot
- Atmospheric mood elements
- Character interaction possibilities
- Symbolic/thematic resonance
Setting context: [Story situation]
```

### Cultural Design
```
Develop a [CULTURE TYPE] with these characteristics:
- Core beliefs: [Basic values]
- Social structure: [Organization]
- Technology level: [Advancement]
- Conflicts with: [Other groups]
- Story role: [Function in narrative]
Create: Rituals, language elements, customs, conflicts
```

## Scene Development Prompts

### Scene Planning
```
Plan a scene with these elements:
- Characters present: [List]
- Setting: [Location and time]
- Scene goal: [What must be accomplished]
- Conflict type: [Internal/external/interpersonal]
- Emotional arc: [Starting and ending emotional state]
- Information revealed: [Plot/character development]
- Scene ending: [How it connects to next scene]
```

### Action Sequence Design
```
Design an action sequence involving [SITUATION].
Characters: [Participants and their capabilities]
Setting: [Environment and constraints]
Stakes: [What's at risk]
Pacing: [Fast/slow moments]
Resolution: [How it ends and consequences]
Focus on: Clear action, character choices, consequences
```

### Emotional Scene Development
```
Write a scene focusing on [EMOTIONAL SITUATION].
Character: [Who experiences the emotion]
Emotion: [Specific feeling and intensity]
Trigger: [What causes this emotion]
Setting: [Where and when]
Other characters: [Who else is present and their role]
Show don't tell: Use actions, dialogue, sensory details
```

## Brainstorming Prompts

### Plot Problem Solving
```
I have a plot problem: [DESCRIBE ISSUE]
Current situation: [Where the story is]
Desired outcome: [Where it needs to go]
Constraints: [What can't change]
Character goals: [What characters want]
Available resources: [Tools/abilities/allies]
Generate 5-10 possible solutions considering character consistency and story logic.
```

### Creative Alternative Generation
```

```

## Usage Guidelines

### Prompt Customization
- Replace bracketed placeholders with specific details
- Adjust complexity based on your needs
- Combine prompts for complex requests
- Iterate and refine based on results

### Quality Control
- Always review AI output for accuracy
- Maintain your creative vision as primary
- Use results as starting points, not final products
- Ensure consistency with established story elements

### Documentation
- Save successful prompt variations
- Note which prompts work best for different tasks
- Update prompts based on experience
- Share effective prompts with other collaborators
