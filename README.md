# 🌲 The Digital Garden

My personal knowledge graph and research engine. An automated Obsidian-to-Web pipeline for cross-linking cognitive science and industrial AI methodology.

## 🎯 Concept

This garden is where raw research meets structured thought. It serves as the living documentation for the **Recursive Intelligence** research pipeline, providing a navigable graph of interconnected notes on:

- **Thinking as a Science**: Methodologies for cognitive workload reduction.
- **Industrial AI**: Constraint-aware modeling and IT/OT integration.
- **Recursive Prompting**: My framework for high-noun-density LLM orchestration.

## 🛠️ The Pipeline

This project uses a custom deployment flow to transform Obsidian markdown vaults into a high-performance static site:

1. **Vault**: Local Obsidian environment with bidirectional linking.
2. **Synchronizer**: Digital Garden Obsidian Plugin.
3. **Engine**: 11ty (Eleventy) + SCSS.
4. **Theme**: Scoped style tokens provided by the `recursive-style-system`.
5. **Deployment**: Vercel / Netlify.

## 🚀 Key Features

- **Backlinks & Graph View**: Visualizing node relationships in real-time.
- **High-Density Search**: Instant indexing for 5,000+ words of research.
- **Design Tokens**: Shared CSS variables ensuring brand consistency with the main platform.
